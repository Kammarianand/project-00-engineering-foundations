# Project 00 - Engineering Foundations

## Overview

This project was created to understand the fundamentals of professional software and data engineering project structure.

The objective was not to build a complete application, but to learn:

- Modular coding
- Project directory structure
- Git & GitHub workflow
- Basic ETL pipeline structure
